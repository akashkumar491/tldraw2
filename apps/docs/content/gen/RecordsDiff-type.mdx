A diff describing the changes to a record.


##### Signature


```ts
type RecordsDiff<R extends UnknownRecord> = {
  added: Record<IdOf<R>, R>
  updated: Record<IdOf<R>, [from: R, to: R]>
  removed: Record<IdOf<R>, R>
}
```
